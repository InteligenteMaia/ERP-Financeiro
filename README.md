## 💻 **Quick Start**

### **Pré-requisitos**

```bash
Node.js 18.x+ | PostgreSQL 14+ | Docker 20+ | Git
```

### **🎯 Instalação Rápida**

```bash
# 1️⃣ Clone o projeto
git clone https://github.com/seu-usuario/erp-financeiro.git

# 2️⃣ Entre na pasta
cd erp-financeiro

# 3️⃣ Configure o ambiente
cp .env.example .env

# 4️⃣ Instale com Docker
docker-compose up -d

# 5️⃣ Acesse o sistema
# 🌐 Frontend: http://localhost:3000
# ⚙️ Backend: http://localhost:3333
# 📚 Docs: http://localhost:3333/api-docs
```

### **🚀 Instalação Manual**

<details>
<summary>Ver instruções detalhadas</summary>

```bash
# Backend
cd backend
npm install
npm run migrate
npm run seed
npm run dev

# Frontend (novo terminal)
cd frontend
npm install
npm start
```

</details>
